Scroll to Section

Scroll to Section leverages the smooth scroll script to implement smooth scrolling to anchor links.

Usage

Add a data-scroll data attribute to an element which when clicked will smooth scroll to its associated anchor element. 

<a data-scroll href="#foo">I am a smooth scroll link</a>

Initialize the script 

smoothScroll.init();

Options and Settings

Pass in options and settings to the init() function 

                                
smoothScroll.init({
    selector: '[data-scroll]', // Selector for links (must be a class, ID, data attribute, or element tag)
    selectorHeader: null, // Selector for fixed headers (must be a valid CSS selector) [optional]
    speed: 500, // Integer. How fast to complete the scroll in milliseconds
    easing: 'easeInOutCubic', // Easing pattern to use
    offset: 0, // Integer. How far to offset the scrolling anchor location in pixels
    callback: function ( anchor, toggle ) {} // Function to run after scrolling
});

You can also override global options and settings with data attributes via HTML, although it’s preferred to set them in JavaScript. 

                                
<a data-scroll
    data-options='{
        "speed": 500,
        "easing": "easeInOutCubic",
        "offset": 0
    }'
    >
   I am a smooth scroll link
</a>

For additional documentation visit the library’s official repository page.
