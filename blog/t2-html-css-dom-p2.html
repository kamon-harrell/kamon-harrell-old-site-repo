What is grid based design?

Grid-based design is the idea that a website should be built around some kind of grid structure, as this accommodates well with the fact that all elements on the web are square or rectangular. Grid based design also helps people eyes follow a page naturally and allows for an organized site.

Whats all the hype about responsive webpages?

The hype of responsive web pages, which is well deserved, is about websites instinctively changing the content of a page, to fit the screen size of whatever device is viewing the content. Basically, this means the content is scalable, in order to appear attractive on a multitude of devices.

What is semantic structure?

Semantic structure refers to the order in which semantic tags such as image, header, and form are placed on the page. this helps to organize the page, and the content, as well as making tagging elements on the page much easier.
